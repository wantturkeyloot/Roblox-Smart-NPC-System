# Roblox-Smart-NPC-System
🧠 Smart NPC AI System for Roblox
💡 Concept

This project is a dynamic NPC behavior system where NPCs react, adapt, and simulate basic decision-making instead of using simple scripted paths.

They can:

React to players
Remember past interactions
Change behavior over time
Simulate basic personality traits
⚙️ Features
🧭 State Machine System

NPC states include:

Idle
Patrol
Suspicious
Chase
Escape
🧠 Memory System

NPCs can remember:

Recent player encounters
Attacks
Sounds or disturbances
🔄 Adaptive Behavior
If a player attacks often → NPC becomes more defensive
If a player hides → NPC investigates hiding spots
🎭 Personality Types
Aggressive
Cowardly
Neutral
